# Vegetation_orchard_Brno

Data and supporting materials associated with the manuscript:

"Spatial patterns of plant and soil microbial communities across row and interrow habitats in apple orchards"

## Repository contents

### data
Contains datasets used for statistical analyses.

### scripts
Contains R scripts used for data processing, statistical analyses, and figure preparation.

## Study description

The study investigated vegetation characteristics and soil microbial communities in orchard floor habitats under different management systems. Vegetation composition, cover, species richness, functional groups, and microbial parameters were evaluated in row and interrow positions.

## Software

Statistical analyses were performed in R.
